HTML : 웹페이지를 만드는 언어
HyperText Markup Language
Text 편집기, 개발도구 등으로 코딩을 하고 Web Browser로 확인하는 과정을 거친다.
확장자는 .html , .htm 으로 구성 된다.

문법
HTML의 문법은 태그로 구성이 되어있다. -> 태그를 알아야 한다.
태그(tag) : HTML 코드에서 정보(콘텐츠)를 정의하는 형식
태그는 <> 와 </> 로 구성되어있고 각각 시작과 끝을 알린다.
태그는 콘텐츠를 감싸며, <> 내부에는 태그명이 들어가고 이는 콘텐츠의 성격과 의미를 나타낸다.

태그 사용 형태)
<태그명> 콘텐츠 기입 </태그명>

단일 태그
태그 중 시작과 끝을 굳이 표현하지 않아도 되는 태그가 있다. 이를 단일 태그라고 한다.
단일 태그 사용 형태)
<태그명> or </태그명> -> 콘텐츠를 감쌀 필요가 없다. -> 시작과 끝을 구분하지 않아도 된다.

속성
태그의 부가적인 기능을 정의. -> 강제되지 않음 (선택사항)
속성은 시작 태그의 내부에서 정의함. 속성의 개수는 제한이 없다.
속성이 있는 태그의 예시)
<태그명 속성명 = "속성값"> 콘텐츠 </태그명> 
<태그명 속성명 = "속성값"/> -> 단일 태그
-> 태그명과 속성명은 space로 구분하고, 속성 값은 ""를 사용해서 구분한다.

주석
다른 프로그래밍 언어에서 사용되는 주석과 같다.
주석의 예시)
<!-- 주석의 내용 -->
