HTML : 웹페이지를 만드는 언어
HyperText Markup Language
Text 편집기, 개발도구 등으로 코딩을 하고 Web Browser로 확인하는 과정을 거친다.
확장자는 .html , .htm 으로 구성 된다.
개발자 도구 : F12

문법
HTML의 문법은 태그로 구성이 되어있다. -> 태그를 알아야 한다.
태그(tag) : HTML 코드에서 정보(콘텐츠)를 정의하는 형식
태그는 <> 와 </> 로 구성되어있고 각각 시작과 끝을 알린다.
태그는 콘텐츠를 감싸며, <> 내부에는 태그명이 들어가고 이는 콘텐츠의 성격과 의미를 나타낸다.

태그 사용 형태)
<태그명> 콘텐츠 기입 </태그명>

단일 태그
태그 중 시작과 끝을 굳이 표현하지 않아도 되는 태그가 있다. 이를 단일 태그라고 한다.
단일 태그 사용 형태)
<태그명> or </태그명> -> 콘텐츠를 감쌀 필요가 없다. -> 시작과 끝을 구분하지 않아도 된다.

속성
태그의 부가적인 기능을 정의. -> 강제되지 않음 (선택사항)
속성은 시작 태그의 내부에서 정의함. 속성의 개수는 제한이 없다.
속성이 있는 태그의 예시)
<태그명 속성명 = "속성값"> 콘텐츠 </태그명> 
<태그명 속성명 = "속성값"/> -> 단일 태그
-> 태그명과 속성명은 space로 구분하고, 속성 값은 ""를 사용해서 구분한다.

주석
다른 프로그래밍 언어에서 사용되는 주석과 같다.
주석의 예시)
<!-- 주석의 내용 -->

HTML 문서의 구조
<!DOCTYPE html>
<html>
    <head>
        <meta charset="utf -8">
        <title>문서의 제목을 쓰는 곳</title>    
    </head>
    <body>
        브라우저 화면에 표시할 내용을 작성하는 곳
    </body>
</html>

<!DOCTYPE html> : 문서의 첫 부분에서 문서 유형을 지정하는 단일 태그
웹 브라우저에게 이 문서가 무엇임을 알려주어 웹 브라우저가 잘 처리하게 해줌

<html> ~ </html> : 문서 유형을 지정한 후 실제 문서가 시작되고 끝나는 것을 나타내는 태그
이 태그 내부에 다양한 태그를 넣어서 문서의 내용을 구성한다.

<head> ~ </head> : 웹 브라우저 화면에는 나오지 않지만 웹 브라우저가 알아야 할 정보들을 포함

<meta charset="utf -8"> : 문자 인코딩 및 문자 키워드 등에 대한 요약 정보를 기입하는 단일 태그
문자 인코딩: 한글을 표시하기 위해 문자 세트를 지정하는 작업 -> 한글 + 영어를 쓰기 위해서는 utf-8 방식 사용 추천

<title> ~ </title> : 문서의 제목을 나타낸다. 콘텐츠는 브라우저 탭에 표시된다.

<body> ~ </body> : 실제 브라우저 화면에 표시되는 내용을 입력하는 태그
태그 내부에 태그를 포함하는 방식으로 콘텐츠를 다양하게 구성 가능
들어가는 내용
-텍스트를 표시하는 태그
-이미지를 표시하는 태그
-각종 사용자 인터페이스(버튼, 입력란, 드롭다운 메뉴 등)를 나타내는 태그

텍스트 태그
문단(paragraph)
p 태그는 문단 요소를 나타내는 태그이다. 가장 많이 사용되는 텍스트 태그
하나의 p 태그는 하나의 문단을 표현 문단과 문단 사이에는 공백이 존재.

제목(headline)
h 태그는 제목 요소를 나타내는 태그이다.
숫자와 함께 사용되며, 숫자가 1일 때 가장 크고 6일 때 가장 작다.

수평선
hr 태그는 수평선을 표시하는 태그이다. 수평선은 주제 변경 또는 내용 구분을 위해 주로 사용된다.
hr 태그는 단일 태그이다. -> 굳이 닫는 태그를 두지 않는다.

HTML 텍스트의 특징
일반적으로 엔터는 줄바꿈을 의미하는 입력이지만, HTML 코드에서는 이를 무시한다. 또한 스페이스를
통한 공백도 한 번씩 밖에 인정되지 않는다.

-> 줄바꿈 태그와 공백문자
줄바꿈 : br 태그
공백을 두 번 이상 표시 : &nbsp; 를 사용

태그의 구분
블록 레벨 요소를 만드는 태그 vs 인라인 요소를 만드는 태그
블록 레벨 요소 : 자기가 속한 영역의 너비를 모두 차지하여 블록을 형성
인라인 요소 : 자기에게 필요한 만큼의 공간만 차지한다.

앞서 배운 h와 p 태그는 블록 태그이다.

strong 태그는 감싸고 있는 콘텐츠를 굵게 표시하는 태그이다. 인라인 요소를 만드는 태그
em 태그는 감싸고 있는 콘텐츠를 기울여 이탤릭체로 표시하는 태그이다. 인라인 요소를 만드는 태그
mark 태그는 감싸고 있는 콘텐츠에 형광펜 표시를 해주는 태그이다. 인라인 요소를 만드는 태그
